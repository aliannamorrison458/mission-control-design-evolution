# 14-Day Rollout Playbook

## Day 1-2: Structure Bootstrap

- Create organization, board groups, boards
- Set board objectives and success metrics
- Enable board defaults (review + approvals)

Exit criteria:
- 4 groups + 7 boards live

## Day 3-5: Agent Topology

- Assign 1 lead + 1 worker per board
- Add role contracts and operating notes
- Seed first execution backlog

Exit criteria:
- Every board has active lead/worker ownership

## Day 6-8: Scheduling Engine

- Apply AI-first scheduling policy
- Add P50/P80 to all active tasks
- Enforce task size and WIP limits

Exit criteria:
- Queue re-ranking is active and visible

## Day 9-11: Monitoring & Control

- Enable heartbeat checks
- Enable daily digest
- Configure escalation thresholds

Exit criteria:
- Heartbeat catches at least one actionable drift
- Digest delivered daily without manual assembly

## Day 12-14: Calibration

- Review KPI trend deltas by group
- Remove low-ROI recurring work
- Tune thresholds and scoring weights

Exit criteria:
- Documented weekly review with policy adjustments
